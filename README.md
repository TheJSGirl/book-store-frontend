## Book Store
- This App is designed to sell and purchase books online.
- It has following functionalities
    - User Authentication
        - Change Password
        - Edit Profile
    - My Books
        - Add Book
        - Delete Book
        - Edit Book
        - Search Bar
    - User Profile
        - Edit Profile
        - Change Password
    
    - Home 
        - All Books
        - Search Bar
        - Add to Cart

## Tech
    - ReactJS
    - Redux
    - CSS
    - Flex Box
